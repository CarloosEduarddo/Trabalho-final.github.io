Introdução

O projeto trata-se de uma padaria online que vende diversas variedades de pães, doces, salgados e outros produtos frescos e de qualidade. A inspiração para tal projeto seria a família.

Objetivos

Os objetivos se resumem em aumentar as vendas, melhorar a visibilidade online e fornecer informações aos clientes.

Público-alvo

Homens ou mulheres entre 20 à 30 anos.

Tecnologias e Plataformas

Foram usados para esse site linguagem html, css e JavaScript nas plataformas Git, GitHub, GitHubDesktop e Visual Studio Code.

Funcionalidades

Algumas foncionalidades deste site são:

  • Compra e venda online;
  • Divulgação de novos produtos.

    • Cronograma

  • 30/02/2024 Adicionar ao site os produtos para a venda.
  • 30/02/2024 Adicionar a página de compra.
  • 31/02/2024 Inicio das vendas.
